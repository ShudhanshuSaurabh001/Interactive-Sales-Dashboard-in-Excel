# 📊 Interactive Sales Dashboard in Excel

## 📌 Project Overview
This project presents an interactive sales dashboard created in Microsoft Excel. The dashboard provides insights into sales performance, profit, product categories, and regional trends using Pivot Tables, Charts, and Slicers for dynamic analysis.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Track profit and quantity sold.
- Compare sales across regions and categories.
- Identify top-performing products.
- Provide an interactive reporting experience.

---

## 📂 Dataset Features

The dataset includes:

- Order ID
- Order Date
- Customer ID
- Customer Name
- Region
- Category
- Product
- Price
- Quantity
- Sales Amount
- Profit
